Etiquetas:

La primera cosa que hay que saber antes de programar un documento son los simbolos esenciales
Los simbolos esenciales son los siguientes:

  • < > dentro de estos simbolos se escribe el comando.
    Ejemplo:
    <html>Ejemplo<html>
  • / este simbolo dentro de estos otros < > significa que la accion a terminado.
    Ejemplo:
    < html > Ejemplo < /html >

Despues de haber visto los simbolos esenciales vamos a ver los comandos basicos y esenciales para hacer cualquier tipo de archivo.
Todo esto dentro de los simbolos que hemos visto anteriormente.
Los comandos son los siguientes:

  1. <!doctype html >
    Este comando se utiliza para señalar que se trata de un archivo html 5 que es la generacion acual de html
    Este comando no tiene cierre con lo cual no existe esto < /!doctype html >
  2. Despues de el doctype va la accion del documento , el cual seria < html > < /html >
    Todo lo que se escriba entre el inicio y el final de esta accion , tanto comandos como escritura seran el archivo en si.
  3. El html se divide en dos troncos principales los cuales serian
    • < head > Ejemplo < /head >
      Este comando sirve para determinar la cabeza y dar una descripcion al archivo. Esto no se ve en la pantalla excepto el titulo.
    • < body > Ejemplo < /body > Como bien indica su nombre es el cuerpo del archivo. Basicamente aqui se escribe lo que luego se mostrara en pantalla.
  4. Dentro de la cabeza ( head ) van los siguientes comandos.
    • < title > Ejemplo < /title >
      Este comando determina lo que va a ser el titulo. El titulo es lo que aparece en la barra que hay encima de la de busqueda.
    • < meta >
      Al igual que el doctype este comando no tiene cierre.
      Este comando sirve para añadir una descripcion del prollecto. Un autor, o las palabras clave por las cuales se podria encontrar esta web.
  5. Dentro del cuerpo ( body ) van los siguientes comandos.
    • < header > Ejemplo < /header >
      Este comando sirve para especificar que hay en la cabecera como bien indica su nombre.
      Este comando es mas para estilizar la pagina que para programarla en si.
    • < nav > Ejemplo < /nav >
      Este comando es parecido al anterior, sirve para indicar la navegacion.
    • < article > Ejemplo < article >
      Este comando sirve para determinar donde estara el articulo de la pagina web.
    • < footer > Ejemplo < footer >
      Al igual que todos los anteriores es mas estetico. Sirve para determinar el pie de pagina.
    • < aside > Ejemplo < /aside >
      Este comando es estetico. Sirve para determinar el lateral de la pagina. Ahi se suelen insertar las imagenes.
    • < p > Ejemplo < /p >
      Este comando sirve para determinar el inicio de un parrafo y el final de este mismo.
    • < br >
      Este comando no tiene principio y fin. Sirve para cambiar de linea.
    • < center > Ejemplo < /center >
      Este comando sirve para centrar un texto.
    • < font > < /font >
      Este comando sirve para cambiar la fuente, basicamente, tamaño, color y tipo de letra.
      Para determinar el tamaño se escribe:
      < font size="x" >
      Para determinar el tipo de letra se escribe:
      < font face="x" >
      Para determinar el color se escribe:
      < font color="" >
      Para el color se utilizan codigos. Asi que aqui os dejpo una web con los codigos de todos los colores.
      Pagina colores html
    • < a > Ejemplo < /a >
      Este comando sirve para insertar enlaces tanto a webs como a otro documento.
      Siempre se escribe de la siguiente forma:
      < a href="enlace" > Ejemplo < /a >
    • < ul > Ejemplo < /ul >
      Este comando sirve para hacer una lista desorganizada(unorganaized list).
      Esta lista esta ordenada con puntos.
    • < ol > Ejemplo < /ol >
      al contrario que el anterior es para hacer una lista organizada(organized list).
      Esta lista esta ordenada con numeros.
    • < li > Ejemplo < /li >
      Este comando es el que se pone dentro de el < ul > y el < ol >

Estos serian las etiquetas mas basicas que existen en html y las necesarias para empezar a programar tu propio html.
