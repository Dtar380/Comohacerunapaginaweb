Aprende los codigos basicos para empezar a crear tu web gratuitamente

  • Etiquetas Basicas html

En esta pagina web te enseñaremos a como crear tu propia pagina web paso a paso sin tener que pagar y totalmente personalizable.
Esto se consigue gracias al lenguage de programacion html en el cual esta basado todo internet.

Para empezar ha entender todo esto te voy ha dictar los siguientes pasos.

  1. Primero haz clic derecho en esta pagina web o en cualquier otra.
  2. Luego de todas las opciones que te aparezcan elige:
    "ver codigo fuente de la pagina"
  3. Entonces te aparecera todo el codigo de esta pagina web.

Si quieres aprender las etiquetas basicos del lenguage de programacion html sube y haz clic en en enlace que pone "Etiquetas basicas html".

Para poder programar el codigo hay muchos softwares pero se puede hacer simplemente desde el bloc de notas aunque recomiendo bastante visual studio code.
